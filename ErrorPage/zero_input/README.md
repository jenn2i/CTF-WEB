# Zero Input

---

## 개요

이 서버는 어떤 사용자에게만 플래그를 보여줍니다.  
그러나 이 서버는 입력을 받지 않습니다.

- 쿼리스트링? 없음.
- POST 요청? 없음.
- input 태그? 없음.

단 하나, 당신의 **쿠키**만이 열쇠입니다.

---

## 힌트

- 쿠키는 자동으로 Set-Cookie 헤더로 설정됩니다.
- 서버는 이 쿠키 값과 비밀 salt를 SHA256 해싱합니다.
- 해시 끝이 '00' 이면 flag가 출력됩니다.
